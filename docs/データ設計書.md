## 1. 型定義（`types/index.ts`）

### 1.1 QuarterKey

四半期を一意に識別する文字列ユニオン型。

```typescript
type QuarterKey =
  | "FY25_1Q" | "FY25_2Q" | "FY25_3Q" | "FY25_4Q"
  | "FY26_1Q" | "FY26_2Q";
```

### 1.2 Quarter

四半期メタ情報。

| フィールド | 型 | 説明 |
|---|---|---|
| `key` | `QuarterKey` | 識別キー |
| `label` | `string` | 表示ラベル（例: "FY25 1Q"） |
| `period` | `string` | 期間テキスト（例: "2024年4〜6月"） |

### 1.3 Question

アンケート質問定義。

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | `string` | 質問ID（スネークケース） |
| `category` | `string` | カテゴリ名（日本語） |
| `text` | `string` | 質問文 |
| `icon` | `string` | 絵文字アイコン |

### 1.4 QuarterData

四半期集計データ（ダッシュボード表示用）。

| フィールド | 型 | 説明 |
|---|---|---|
| `quarter` | `QuarterKey` | 四半期キー |
| `label` | `string` | 表示ラベル |
| `period` | `string` | 期間テキスト |
| `totalScore` | `number` | 総合スコア（0〜100） |
| `turnoverRate` | `number` | 離職率（%） |
| `respondents` | `number` | 回答者数 |
| `categoryScores` | `Record<string, number>` | カテゴリID → スコア（0〜100） |

### 1.5 SurveySubmission

個別回答データ（回答一覧画面用）。

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | `string` | 回答ID |
| `employeeId` | `string?` | 従業員ID |
| `employeeName` | `string?` | 氏名 |
| `department` | `string?` | 部署名 |
| `quarter` | `QuarterKey` | 回答四半期 |
| `submittedAt` | `string` | 回答日（YYYY-MM-DD） |
| `answers` | `SurveyAnswer[]` | 各質問の回答 |
| `totalScore` | `number` | 総合スコア（0〜100） |

### 1.6 ScoreLevel

スコアレベル判定結果。

```typescript
type ScoreLevel = "good" | "caution" | "warning" | "danger";
```

## 4. スコアリングロジック（`lib/scoring.ts`）

```
正規化スコア = Σ(各質問の回答値) / 最大合計点 × 100
最大合計点 = 質問数(10) × 最大スコア(5) = 50

レベル判定:
  score >= 80  → "good"    （#16A34A）
  score >= 60  → "caution" （#D97706）
  score >= 40  → "warning" （#EA580C）
  score < 40   → "danger"  （#CC0000）
```
