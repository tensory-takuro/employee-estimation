### SCR-04: 管理者ダッシュボード（`/admin`）

**レイアウト**: サイドバー固定（260px）+ メインコンテンツ

```
┌───────────┬────────────────────────────────────────┐
│ [紺]      │  [BarChartアイコン] ダッシュボード      │
│ ES Manager│                                        │
│ ──────── │  ┌─────────────────────────────────┐  │
│ ダッシュ  │  │ 四半期セレクター                  │  │
│ 回答一覧  │  │ [FY25 1Q]■ [FY25 2Q]■ ...      │  │
│ ──────── │  └─────────────────────────────────┘  │
│ 従業員画面│                                        │
│           │  [KPI] [KPI] [KPI] [KPI]              │
│ © 2026   │                                        │
│           │  ┌──────────────┐ ┌──────────────┐   │
│           │  │積み上げ棒グラフ│ │折れ線グラフ  │   │
│           │  │カテゴリ別推移 │ │満足度+離職率 │   │
│           │  └──────────────┘ └──────────────┘   │
│           │                                        │
│           │  ┌─────────────────────────────────┐  │
│           │  │  カテゴリ別スコアテーブル         │  │
│           │  └─────────────────────────────────┘  │
└───────────┴────────────────────────────────────────┘
```

**四半期セレクター**:
- 全四半期を個別チェック（トグルボタン形式）
- 「すべて選択」リンク
- 選択変更でグラフ即時更新（クライアントサイドフィルター）

**KPIカード（4枚）**:
- 直近の満足度スコア（色コーディング）
- 前四半期比（+/-pt）
- 直近の離職率（%）
- 回答者数（名）

**積み上げ棒グラフ**:
- X軸: 四半期ラベル
- Y軸: スコア（0〜100）
- 各バー: 10カテゴリを色分けして積み上げ
- ツールチップ: カテゴリ別スコア + 平均値

**折れ線グラフ**:
- 左Y軸: 満足度スコア（0〜100）
- 右Y軸: 離職率（0〜15%）
- 参照ライン: 目標スコア70（ゴールド破線）
- 最終データポイントを強調（大きめドット）
